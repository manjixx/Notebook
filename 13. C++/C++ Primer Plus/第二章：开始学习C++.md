### 2.1.1 main 函数

> 1. 作为接口的函数头

该函数头描述的是 `main()` 和操作系统之间的结偶

```c++
/**
    返回类型
    函数名
    形参列表
 */
int main()
```

> 2. 为什么 main() 不能使用其他名称

运行C++程序时，通常从 `main()` 函数开始执行。
